## What We Build

We create open-source tools that make managing GitHub issues smarter and more efficient:

- **Semantic Search** — Find related issues using AI, not just keywords
- **Duplicate Detection** — Automatically identify similar issues
- **Intelligent Routing** — Route issues to the right repository
- **Smart Triage** — AI-powered labeling and classification
